RTP-TS Offset Calculator
========================
========================

This small app can calculate the offset between the RTP timestamp and the receiving timestamp of an RTP package

No need to install anything, just open the index.html with your webbrowser (Only tested on Firefox)
For convenience copy the Shortcut to your preferred directory to directly open the tool.


=======USAGE=======

Copy the RTP timestamp of the RTP package to the fiels called: RTP timestamp
Copy the receiving timestamp to the field called: Receiving timestamp

In this directory there is a screenshot which shows the needed headers
